# AIのアルゴリズム
まず、盤面の見方は二通りある:
  1) 盤面をAIが駒を動かす直前であると見る
  2) 盤面をユーザーが駒を動かす直前であると見る
ある盤面が与えられたとして、これをAIが駒を動かす直前であると見ることにしよう。  
お互いが非常に賢いプレイヤーであるとすると、この盤面はすでに勝敗が決定している。  
これは、ニュートリーコが「完全情報ゲーム」であるためである。  
そこで、AIが勝つことが確定している状態を全て探し尽くしてしまおう、という戦略を取る。
